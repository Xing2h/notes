# 1、JavaScript基础

## 1、字面量

### 1、数字字面量

数字字面量，可以是：整数、小数、科学计数法。

```javascript
1234
1.23
123e5(12300000)
```

### 2、字符串字面量

字符串字面量可以用单引号，也可以用双引号。

```javascript
"String"
'String'
```

### 3、表达式字面量

```js
1+2
x * y //(x,y要先赋值)
```

### 4、数组字面量

```js
[1,2,3,4]
```

### 5、对象字面量

```js
{firstName:"Jhon", lastName:"Doe",age:19}
```

### 6、函数字面量

```js
function functionName(//参数){//函数体}
```

## 2、变量

JavaScript中用`var`来定义变量

```js
var a,b;
var x = 1;
```
